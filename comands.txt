comandos durante o curso

install yarn
-> npm i -g yarn

inicializar um projeto com yarn
-> yarn init -y

install TYPESCRIPT com DEV
-> yarn add -D typescrit

config do typescrit
-> yarn tsc --init

install esLint
npm init @eslint/config

install node express
yanr add express

install types node express
yarn add -D @types/express

install dependecies de dev para build e runnig o server
yarn add -D ts-node
yarn add -D nodemon

install container mongoDB no docker
docker run --name mongo -p 27017:27017 -d mongo

para estabelecer a conexão e munipulação dos dados com o mongoDB deve instalar
yarn add mongoose

para manipulacao de arquivos em rotas
yarn add multer
yarn add -D @types/multer
